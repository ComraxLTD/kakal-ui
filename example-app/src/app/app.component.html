Simple Autocomplete example Set autocomplete ui 1. create an AutocompleteQuestionModel, or a Question Model with controlType property set to autocomplete 2. import FormService 3. call the method getFiledControl() to create new FormControl instance 4. create kkl-form-autocomplete in the html template 5. pass the question and the control as inputs Set options 1. options array should be in the format of SelectOption interface 2. after subscribe inject the array to the question model
